## 前端啟動

```powershell
cd frontend
npm install
npm run dev
```

預設網址：

```text
http://localhost:3000
```

前端 build 驗證：

```powershell
cd frontend
npm run build
```

## 後端啟動

```powershell
cd backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
python app.py
```

Health check：

```text
GET http://127.0.0.1:5000/health
```

預期回應：

```json
{ "status": "ok" }
```

## DATABASE_URL

請使用環境變數管理 PostgreSQL 連線：

```text
DATABASE_URL=postgresql://USER:PASSWORD@HOST:PORT/DATABASE
```

開發階段若已使用 Aiven PostgreSQL，可暫時保留；免費部署階段規劃改用 Neon PostgreSQL Free。程式碼不應硬編碼資料庫帳密或供應商資訊。

## Demo 操作流程

1. 啟動前端。
2. 開啟首頁 Dashboard。
3. 檢查三個核心區塊：所有熱門話題、符合產業熱門話題、自訂關鍵字熱門話題。
4. 點擊任一話題卡片。
5. 在詳情彈窗檢查 Topic、Score、Momentum、Summary、Insight、Source、Content Count。
6. 檢查 Related Content、Inspiration Ideas、Platform Tags、Topic Tags。
7. 檢查 AI 短摘要、AI 完整摘要與爭議提醒。
8. 選擇語氣與發文角度。
9. 點擊產生文案。
10. 確認同一話題最多保留最新三版文案。
