環境變數：

- `AI_PROVIDER=mock`：預設使用 mock / rule-based AI
- `AI_PROVIDER=openai`：嘗試使用 OpenAI adapter
- `OPENAI_API_KEY`：OpenAI API Key
- `OPENAI_MODEL`：OpenAI model，預設 `gpt-4o-mini`

fallback 規則：

- 未設定 `AI_PROVIDER=openai` 時使用 mock fallback
- 未設定 `OPENAI_API_KEY` 時使用 mock fallback
- OpenAI API 回傳錯誤時使用 mock fallback
- 前端呼叫 `/api/ai-analysis` 失敗時使用本地 mock fallback

安全原則：

- 不移除 mock / rule-based AI
- 不在 client component 暴露 API Key
- Demo 未設定 OpenAI API Key 時仍可正常展示
- 真實 AI 失敗不影響話題詳情與文案生成流程
