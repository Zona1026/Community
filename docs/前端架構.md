# 前端架構

本文件描述前端架構與資料流，不維護目前狀態。

目前專案狀態請以 `docs/CURRENT_STATUS.md` 為準。技術選型請以 `docs/技術資源與選型.md` 為準。

## 技術選型

- Next.js。
- React。
- TypeScript。
- CSS：`frontend/src/app/globals.css`。

## 主要目錄

```text
frontend/
├─ src/
│  ├─ app/
│  │  ├─ api/
│  │  ├─ page.tsx
│  │  ├─ layout.tsx
│  │  └─ globals.css
│  ├─ components/
│  ├─ data/
│  ├─ lib/
│  │  ├─ ai/
│  │  ├─ api/
│  │  ├─ dashboard/
│  │  ├─ import/
│  │  └─ reddit/
│  └─ types/
└─ package.json
```

## 前端資料流

```text
JSON / MockData / Reddit
↓
ContentItem
↓
NormalizedContent
↓
TopicCluster
↓
TrendSignal
↓
TrendReport
↓
InspirationIdea
↓
Dashboard UI
```

## 主要畫面與元件

- 首頁 Dashboard。
- Topic card。
- Topic detail modal。
- Source Analysis。
- User Settings。
- Favorites。
- AI analysis / generated copy UI。

## AI Provider

前端 AI 呼叫透過 provider / fallback 架構：

- Mock / rule-based fallback。
- OpenAI adapter。
- Gemini adapter。

未設定 key 或 provider 失敗時，應保留 fallback。

## 啟動方式

```powershell
cd frontend
npm install
npm run dev
```

## 建置驗證

```powershell
cd frontend
npm.cmd run build
npx.cmd tsc --noEmit
```
