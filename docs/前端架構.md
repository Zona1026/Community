# 前端架構

## 產品定位

前端主頁應是「趨勢雷達 / Trend Radar」導向。

本產品不是社群文案產生器，也不是讓使用者一進來輸入品牌並產生貼文。

主要流程：

1. 發現趨勢
2. 理解趨勢
3. 評估是否適合自己的產業
4. 產生內容靈感
5. 再由使用者決定是否延伸成實際貼文

## 技術選型

- Vue 3
- Bootstrap
- Vue Router
- Axios

## 導覽列 Navigation

主選單：

- 趨勢雷達
- 趨勢探索
- 產業靈感
- 趨勢報告
- 收藏

避免使用：

- 文案產生器
- AI 寫社群文
- 一鍵生成文案

## 首頁 Hero

主標題：

```text
發現正在升溫的內容趨勢
```

副標：

```text
即時追蹤社群、新聞與網路討論，找出值得跟進的話題、梗與內容靈感。
```

主要 CTA：

- 查看今日趨勢
- 輸入產業找靈感

避免 CTA：

- 立即產生貼文
- AI 幫你寫社群文
- 一鍵生成文案

## 趨勢總覽 Dashboard

首頁包含以下區塊，第一版可先使用 mock data。

### 今日熱門趨勢

依 Heat Score 排序，顯示目前最值得關注的趨勢。

### 快速升溫趨勢

依 Growth Rate 與 Momentum 顯示正在升溫的話題。

### 跨平台爆紅趨勢

顯示同時在多個平台出現訊號的趨勢。

### 即將過氣 / 熱度下降趨勢

顯示 Momentum 為 cooling 或 Lifecycle Stage 為 declining、expired 的趨勢。

### 編輯精選 / AI 推薦觀察

顯示適合觀察、低風險或具內容延伸潛力的趨勢。

## 趨勢卡片 Trend Card

每張卡片至少顯示：

- Topic 名稱
- 一句話摘要
- Heat Score
- Growth Rate
- Lifecycle Stage：萌芽 / 成長 / 爆紅 / 衰退 / 過氣
- 主要來源平台，例如 Threads、Instagram、News、Dcard、PTT
- 相關關鍵字
- 可切入產業
- 風險提示，例如：爭議高、快過氣、不適合硬跟

卡片 CTA：

- 查看趨勢詳情
- 產生內容靈感

## 趨勢詳情頁 Trend Detail Page

路徑：

```text
/trends/:id
```

詳情頁包含：

- 趨勢名稱
- 趨勢摘要
- 為什麼正在紅
- 代表內容 / 代表貼文列表
- 相關關鍵字
- 跨平台表現比較
- Heat Score
- Growth Rate
- Momentum
- Lifecycle Stage
- 適合切入的產業
- 不適合硬跟的產業
- 可能的內容角度
- 風險與注意事項
- 過氣風險判斷

## 內容靈感產生區 Inspiration Panel

這個功能不是直接產生完整社群文，而是協助使用者把趨勢轉成內容方向。

表單欄位：

- 我的產業
- 品牌調性
- 目標受眾
- 想發布的平台
- 想要的內容形式，例如短影音、圖文、限動、長文、電子報

輸出內容：

- 可以怎麼切入這個趨勢
- 內容角度
- 標題方向
- 開場 hook
- 適合平台
- 不建議的切法
- 需要注意的風險

## 主要頁面

路徑：

```text
/
```

包含 Hero 與 Dashboard。

### 趨勢探索

路徑：

```text

```

路徑：

```text

```

使用者輸入產業後，取得可借勢話題。

### 趨勢報告

路徑：

```text
/reports
```

彙整趨勢摘要、跨平台訊號、生命週期與觀察建議。

### 收藏

路徑：

```text
/saved
```

收藏想持續追蹤的趨勢、報告或靈感。

## 前端型別

核心命名：

- Content = 原始內容
- Topic = 很多內容討論的同一個主題
- Trend = 正在快速變熱的 Topic
- Signal = 判斷趨勢的證據
- Insight / Inspiration = 使用者最後真正需要的洞察與靈感

前端 type/interface：

- ContentItem
- Topic
- Trend
- TrendSignal
- TrendReport
- InspirationIdea

## 視覺方向

整體風格應像「趨勢雷達 / 內容研究工具」，不是文案產生器。

建議使用：

- dashboard layout
- trend card
- score badge
- lifecycle badge
- platform chips
- growth indicator
- 風險提示
- 可點擊的詳情頁或詳情抽屜

## 驗收重點

- 首頁第一眼是 Trend Radar，而不是文案工具。
- Hero 有「查看今日趨勢」與「輸入產業找靈感」。
- Dashboard 有五個趨勢區塊。
- Trend Card 顯示分數、成長率、生命週期、平台、產業與風險。
- Trend Detail Page 能完整理解趨勢。
- Inspiration Panel 產生內容方向，不直接輸出完整貼文。
