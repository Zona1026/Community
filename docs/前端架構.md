# 前端架構

## 目前狀態

前端正式採用 Next.js + React + TypeScript，不再使用 Vue 3、Vue Router 或 Axios 作為 Phase 1 前端基礎。

Phase 1 Demo 已完成，目前進入 Week 11 展示準備階段。

## 技術選型

- Next.js
- React
- TypeScript
- CSS：`frontend/src/app/globals.css`

## 主要目錄

frontend/
