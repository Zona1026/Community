# 開發資源清單

本文件記錄目前專案實際使用或未來可能使用的開發資源。Phase 1 Demo 已完成，目前進入 Week 11 展示準備階段。

## 目前實際使用

### Next.js

- 用途：前端 App Router、首頁 Dashboard、Phase 1 Demo UI
- 使用位置：`frontend/`
- 官方文件：https://nextjs.org/docs

### React

- 用途：前端元件、狀態管理、詳情彈窗互動
- 使用位置：`frontend/src/components/`
- 官方文件：https://react.dev/

### TypeScript

- 用途：前端型別、資料流程型別定義
- 使用位置：`frontend/src/types/`
- 官方文件：https://www.typescriptlang.org/docs/

### Flask

- 用途：後端骨架、health check、JSON sample reader
- 使用位置：`backend/`
- 官方文件：https://flask.palletsprojects.com/

### PostgreSQL

- 用途：正式規劃資料庫
- 狀態：Phase 1 尚未正式寫入
- 官方文件：https://www.postgresql.org/docs/

### DATABASE_URL

- 用途：統一管理 PostgreSQL 連線
- 使用位置：`backend/.env.example`
- 注意：不得硬編碼資料庫帳密、Host 或供應商資訊

## 部署與資料庫候選

### Aiven PostgreSQL

- 用途：開發階段若已建立可暫時保留
- 狀態：不是唯一綁定供應商

### Neon PostgreSQL Free

- 用途：免費部署階段候選 PostgreSQL
- 狀態：正式部署前再設定
