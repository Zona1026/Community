## JSON Mock Reader

`backend/app.py` 內已提供 `load_json_mock_data()`，可讀取前端 JSON sample。

目的：

- 驗證後端可以讀取展示資料
- 保留未來正式 API 串接位置
- 不寫入資料庫

## DATABASE_URL

本專案正式規劃資料庫為 PostgreSQL，後端連線統一使用：

```text
DATABASE_URL=postgresql://USER:PASSWORD@HOST:PORT/DATABASE
```

開發階段若已使用 Aiven PostgreSQL 可暫時保留；免費部署階段規劃改用 Neon PostgreSQL Free。程式碼不得硬編碼資料庫帳密、Host 或供應商。
