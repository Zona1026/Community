# 技術資源與選型

本文件是技術選型與開發資源的 Single Source of Truth。

舊文件 `docs/技術使用清單.md` 與 `docs/開發資源清單.md` 已合併至此；舊版內容已封存在 `docs/archive/`。

## 已採用

| 類別 | 技術 / 資源 | 使用位置 | 說明 |
|---|---|---|---|
| Frontend | Next.js | `frontend/` | App Router、API routes、Demo UI |
| Frontend | React | `frontend/src/components/` | Dashboard、topic card、detail modal |
| Frontend | TypeScript | `frontend/src/` | 型別與資料流程 |
| Frontend | CSS | `frontend/src/app/globals.css` | 目前 UI 樣式 |
| Backend | Python | `backend/` | Flask 後端 |
| Backend | Flask | `backend/app.py` | Health check、topics、settings、favorites API |
| Database | PostgreSQL | `backend/schema.sql` | 正式資料庫方向 |
| Database | `DATABASE_URL` | `backend/.env.example` | 統一管理資料庫連線 |
| AI | Mock / rule-based AI | `frontend/src/lib/ai/` | Demo fallback 與穩定展示 |
| AI | OpenAI adapter | `frontend/src/lib/ai/openAiAdapter.ts` | 真實 AI provider 候選路徑，需 API key |
| AI | Gemini adapter | `frontend/src/lib/ai/geminiAiAdapter.ts` | 真實 AI provider 候選路徑，受 quota 影響時 fallback |

## 候選

| 類別 | 技術 / 資源 | 用途 | 備註 |
|---|---|---|---|
| Database hosting | Neon PostgreSQL Free | 免費部署階段候選 PostgreSQL | 正式部署前再決定 |
| Database hosting | Aiven PostgreSQL | 開發階段若已建立可暫時保留 | 不綁定為唯一供應商 |
| AI | OpenAI Platform | 真實 AI 摘要、分析、文案生成 | 需有效 `OPENAI_API_KEY` |
| AI | Gemini API | 真實 AI 摘要、分析、文案生成 | 需有效 `GEMINI_API_KEY` 與可用 quota |
| Export | DOCX / PDF library | Phase 5 匯出 | 待功能進入時再選型 |
| Chart | Chart library | Trend Chart MVP | 尚未導入 |
